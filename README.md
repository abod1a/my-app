# AI Document Analyzer 📄🤖

A smart web application built using **Streamlit** and **Groq API**, allowing users to upload files (PDF, DOCX, TXT) and chat with them to extract information and perform content analysis.

[![Streamlit App](https://static.streamlit.io/badges/streamlit_badge_white_black.svg)](https://my-app-fqv5gpwudvbyqrua5zpavt.streamlit.app/)

## 🌟 Key Features
* **Multi-format Support:** Seamlessly upload PDF, Word, or plain text files.
* **Intelligent Analysis:** Accurate information extraction and content analysis using advanced LLMs.
* **User-Friendly UI:** A clean, intuitive interface powered by Streamlit.
* **High Performance:** Low-latency inference powered by Llama models via the Groq API.

## 🛠 Technologies Used
* **Python:** Core programming language.
* **Streamlit:** Framework for the web application UI and backend operations.
* **Groq API:** Providing access to high-performance AI models (Llama 3.3).
* **PyPDF & Python-docx:** Libraries used to process and extract text from various file formats.
